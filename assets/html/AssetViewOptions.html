Asset View Options

Personalize the assets view on the dashboard depending upon the detailing required on the asset.



You have the option to view the assets as Small Thumbnails, Large Thumbnails or in Detailed Format.


Icon

Description

Small Thumbnails

Large Thumbnails

Detailed/ column Format




You can switch to smaller thumbnail view or the detailed view by choosing the required change view option, from the above shown options.

Smaller Thumbnail view

As seen below, in this view, 8 assets are displayed in a single row and 30 assets are listed on a page.


Detailed view

In this view, the asset along with the asset details and other available options can be viewed (based on user role)

Viewing multi Primary files thumbnails:

The total number of prime files in the asset set is mentioned on the dashboard and the assets thumbnails can be viewed by scrolling.


Multiversion assets are displayed with the number of versions in the lower right corner and arrows on either side so that user can view the other versions.
