Edit Asset Details from Preview

You can edit the existing information or add new properties to existing fields (for example, click the media type dropdown and check/uncheck to select/deselect a media option). Every time an asset is viewed/ edited the details are overwritten, and the changes can be tracked in the asset activity logs.



Name:        Descriptive name given to the asset based on naming your conventions.

Code:        In order to keep the track, assets are required to be tagged with unique code or numbers. 

Media Type:        Media describes the overall categories of the assets, such as images, audio and video.

Category:        Assets are broadly grouped/classified into various categories like Auto, Dining & Entertainment, Grocery, Real estate etc.

Asset Types: An Asset Type is more akin to a label or identification such as Shell Ad, Template, Advert, Image, Logos, Publication, Masthead, Poster etc.

Tags:        Tags are keywords or labels that are attached to the asset to enable searches to find that content and related content        

Description: A detailed account of the certain or salient aspects, characteristics, or features of the asset

To Edit Asset Information

Edit the Media Type, Asset Type, Category, add Tags and description to the selected asset for better asset organization and improved searchability.
