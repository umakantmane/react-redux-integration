Search Assets

Minimize the time it takes to make your assets more searchable. Search is incredibly fast and intuitive and user friendly, when you enter the search word, you will receive automatic prompts for relevant keywords based on similar images and the custom metadata already in use.

Use the search bar to add keywords, add search filters to narrow your search by combining different search criteria from account specific search filters like categories, types or tags.You can save this combination of filters set so that you can easily apply it the next time you are looking for the same assets.


Note: While entering the search keyword in the search bar, relevant suggestions list below, you can select the most appropriate keyword from the suggestions.


  1. Enter the keywords in the Search Bar





  1. To further refine the search criteria, add categories, asset types and/or tags
  • Click the ‘Category’ dropdown and check the required options
  • Click the ‘Asset Types’ dropdown and check the required options
  • Click the ‘Tags’ dropdown and select from check the required options

  • The selected options display above the search results
